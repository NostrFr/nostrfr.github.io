1. Présentation

Nowhr est un générateur souverain de pages statiques. Il propose plusieurs modèles prêts à l’emploi pour créer des micro‑apps : forum, événement, pétition, boutique, fundraiser, message, drop, art.

Aucune inscription. Aucun backend. Aucune base de données. Le serveur ne sert que des fichiers statiques : toute la logique est exécutée dans ton navigateur.

Tu choisis un modèle, tu remplis un formulaire minimal, tu publies, et tu obtiens une URL autonome.

2. Pourquoi c’est intéressant

Nowhr est conçu pour un web souverain, simple et sans dépendances :

  • 100% client‑side : génération locale des pages
  • Aucun backend : pas de base de données, pas d’API
  • Aucun compte : aucune inscription
  • Aucune collecte : pas de cookies, pas d’analytics
  • Modèles multiples : forum, event, fundraiser, store, petition, message, drop, art
  • Instantané : remplir → publier → partager
  • Minimalisme : HTML brut, JS minimal, CSS sobre

C’est un outil pour créer des pages utiles, rapides et souveraines, sans infrastructure.

3. Fonctionnement

3.1 Client‑side uniquement

Nowhr fonctionne entièrement dans le navigateur. La page finale est générée localement, puis publiée sous forme autonome.

3.2 Modèles disponibles

Chaque route /create/... correspond à une micro‑app spécialisée :

  • Forum : /create/forum
  • Event : /create/event
  • Fundraiser : /create/fundraiser
  • Store : /create/store
  • Petition : /create/petition
  • Message : /create/message
  • Drop : /create/drop
  • Art : /create/art

3.3 Publication instantanée

Remplir un formulaire minimal → publier → obtenir une URL. Aucune donnée n’est stockée côté serveur.

3.4 Code minimaliste

HTML simple, JS minimal, CSS brutaliste. Aucune dépendance externe. L’objectif est la souveraineté, pas la complexité.

4. Cas d’usage

  • Forum minimal : espace de discussion sans compte
  • Page d’événement : annoncer une rencontre ou un meetup
  • Fundraiser souverain : collecte de fonds sans intermédiaire
  • Mini‑boutique : lister quelques produits avec liens externes
  • Pétition : texte à signer sans plateforme centralisée
  • Message public : publier un texte en une URL
  • Drop : alternative souveraine à Pastebin
  • Page d’art : présenter une image ou un visuel

5. Place dans l’écosystème souverain

Nowhr s’intègre naturellement dans un stack minimaliste :

  • Textarea.my — écrire
  • Nowhr — générer des micro‑apps statiques
  • HostedNowhere / Drop — publier du texte ou du HTML brut
  • Ditto.pub — diffuser des capsules Nostr
  • Oracolo — éditorialiser et publier des articles
  • Nostr — identité et diffusion
  • Mostro — échanges P2P
  • NostrMail — email souverain

Nowhr est la brique “génération de contenu structuré” du web souverain.
